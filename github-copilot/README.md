# GitHub Copilot & AI Models

This directory contains notes on different AI models and their usage patterns for coding assistance.

## Available Models

- [Claude](claude.md) - Anthropic's Claude models (Sonnet, Opus, Haiku)
- [Gemini](gemini.md) - Google's Gemini models
- [GPT](gpt.md) - OpenAI's GPT models (ChatGPT, GPT-4, etc.)
- [GitHub Copilot](copilot.md) - GitHub's Copilot coding assistant

## General Tips

- Be specific and provide context in your prompts
- Break down complex tasks into smaller steps
- Include relevant code snippets and error messages
- Specify the programming language and framework
- Iterate on prompts based on the responses you get
